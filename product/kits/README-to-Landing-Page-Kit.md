# README to Landing Page Kit

Price: $10

Turn a technical README into a compact landing page.

## Page Formula

1. Product name.
2. One-sentence outcome.
3. Screenshot, terminal block, or demo asset.
4. Three proof points.
5. Direct action button.
6. Short FAQ or usage notes.
