# One Hour Launch Plan

Price: $10

Launch a tiny product in one focused hour.

## Minute 0-10: Pick

Choose one painful micro-problem:

- finding bounties
- writing PR descriptions
- setting up a static checkout
- reviewing AI-generated code
- turning README into sales copy

## Minute 10-30: Build

Create one Markdown file with:

- title
- who it is for
- checklist or templates
- examples
- closing instructions

## Minute 30-45: Package

- create ZIP
- write landing page
- add wallet or payment link
- add download button

## Minute 45-60: Publish

- push to GitHub Pages
- post link in one relevant place
- save the source files
- write a short changelog

## Launch Post

```text
I made a small $10 kit for <audience>.
It helps with <specific outcome>.
Link: <url>
```
