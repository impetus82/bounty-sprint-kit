# No-KYC Checkout Checklist

Price: $10

Use this before setting up a tiny digital-product checkout.

## Payment Options

Lowest friction:

- direct crypto wallet
- PayPal link
- Ko-fi with PayPal or Stripe
- itch.io direct payments
- manual Telegram delivery after proof of payment

## Crypto Checkout Copy

```text
Send $10 USDT TRC20 to:
<wallet>

Then download the file below. This is an honor-system checkout.
```

Manual delivery version:

```text
Send $10 USDT TRC20 to:
<wallet>

Then message the transaction hash to:
<contact>
```

## Risk Notes

- Never ask buyers for seed phrases or private keys.
- Never share your own seed phrase.
- Use a public wallet address only.
- Some exchanges may require KYC to cash out crypto later.
- Card payments almost always involve a processor with compliance rules.

## Minimal Store Checklist

- one product title
- one sentence promise
- three bullets showing what is inside
- wallet address
- network
- price
- download or contact button
- refund note if you want one
- local backup of the file

## Buyer Reply Template

```text
Thanks. I received your transaction hash.

Here is the download:
<link>
```
