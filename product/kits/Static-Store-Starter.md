# Static Store Starter

Price: $10

A minimal plan for selling a digital file from a static page.

## File Tree

```text
store/
  index.html
  styles.css
  product/
    download.zip
```

## Checkout Block

```html
<section>
  <h2>Get the product</h2>
  <p>Send $10 USDT TRC20, then download the ZIP.</p>
  <p><strong>Wallet:</strong> YOUR_ADDRESS</p>
  <a href="product/download.zip">Download ZIP</a>
</section>
```

## Copy Template

```text
This is an honor-system checkout.
If you find the product useful, send $10 before downloading.
```
